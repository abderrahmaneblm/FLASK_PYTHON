Le code QR en pédagogie

Le codes QR envahit notre environnement. Parfois appelé Flashcode , il s’agit d’un code-barre de forme carrée qu’il suffit de scanner, (de "flasher", comme on dit !) avec un smartphone   ou une tablette pour lire les informations qu’il contient. La lecture déclenche alors sur l’appareil une action qui peut être l’affichage d’un texte, la création d’un nouveau contact, l’ajout d’un événement à l’agenda, l’envoi d’un courriel  , l’accès direct à une ressource web de nature variée (image, fichier texte, audio ou vidéo)... [1]

Il fait aujourd’hui son entrée dans le domaine de l’éducation ! Utilisé généralement pour enrichir les documents mis à la disposition des élèves (documents distribués, affichages, vidéoprojections...), le code QR   peut être associé à de multiples situations pédagogiques.

Un apprentissage nomade

De plus en plus, le cours s’effectue en dehors d’une classe dite traditionnelle.
La présence de code QR   offre dans cette perspective une fonctionnalité très intéressante pour la pédagogie :

  • lors d’une exposition, le code permet d’enrichir l’affichage de ressources complémentaires, notamment des vidéos ;
  • lors d’une course d’orientation, le code sert à géolocaliser un lieu ou un parcours ;
  • au cours d’un jeu de piste, le code accessible à chaque étape donne des indices ou propose une activité dont la réussite permet de passer à l’étape suivante ( voir une expérimentation de jeu de piste)
  • ...
    Par ailleurs, la présence de codes QR dans les supports de cours permet aux élèves d’accéder facilement à une page web ou un document pour faire leurs recherches et/ou approfondir le cours depuis n’importe où, n’importe quand.

Un accès aux ressources facilité : des ressources directement intégrées dans l’appareil mobile

L’utilisation des codes QR rend directement accessibles les documents et les pages web dont les liens peuvent être longs ou compliqués à saisir au clavier d’un smartphone   ou d’une tablette. Ainsi, on a l’assurance que les élèves disposent de la bonne ressource. De plus, cette dernière est accessible dans ou en dehors de la classe. Il peut s’agir d’un document, d’une page web à consulter, mais aussi d’une activité à effectuer individuellement ou collectivement (si le lien pointe vers une plateforme collaborative de type Framapad ou Google drive).

Par cette méthode, on peut envisager la diffusion des "devoirs à faire" sous la forme de code QR. Ainsi, pas d’erreur de saisie possible et les données sont automatiquement enregistrées dans l’agenda de l’appareil des élèves.

L’utilisation du cloud   est de plus en plus répandue dans la pratique enseignante pour la facilité d’accès de stockage et de partage des documents. Désireux de partager avec ses élèves un document stocké dans son cloud (Owncloud, Dropbox, Skydrive, Google drive...), l’enseignant peut le faire via un code QR. L’accès en est facilité et le document en question est directement contenu sur leur appareil mobile. Cette méthode autorise l’enseignant à réactualiser le document sans modifier le code communiqué (exemple ).

Un complément de ressources, dans un premier temps caché

Dans certaines situations, le code QR peut servir de point d’étape obligatoire. Pour accéder à la ressource ou l’information qu’il contient, les élèves devront passer par le décodage du code qui mobilise leur appareil. L’enseignant peut utiliser cette opportunité pour gérer les temps d’apprentissage, en régulant l’utilisation des appareils ou en choisissant de faire apparaître le code QR au moment opportun en faisant apparaître un objet, un document sur lequel est inséré le code QR ou par projection du code sur tableau.

Par ailleurs, cette méthode peut être appliquée aux tâches complexes. Les indices, les "coups de pouce", les aides ciblées (aides méthodologiques, protocoles, fiches techniques, ressources documentaires...) sont proposés sous forme de codes QR. Chaque élève les utilise s’il le juge nécessaire. Il adopte une démarche personnelle de résolution pour réaliser la tâche.

Une individualisation du cours

Individualiser, c’est prendre en compte l’hétérogénéité des élèves.
L’accès aux ressources peut aussi être pensé comme un moyen d’aider, d’accompagner les élèves en difficulté (exemple). Proposer aux élèves, sur un même document, d’accéder à des ressources complémentaires via un code QR permet de répondre à un besoin individuel et compenser une difficulté (fichier audio du texte présenté, définitions de termes, indices).

Un environnement d’apprentissage facilité

L’utilisation de codes QR, par la convergence de ressources très diversifiées, participe à la création d’un environnement d’apprentissage. La fusion sur un même support d’une grande quantité de contenus jusque là disséminés ouvre des voies intéressantes en matière de pédagogie.
Intégrer les codes QR sur un même support permet de faire un lien étroit avec et entre les ressources concernées. Le contenu des codes QR peut renvoyer à une diversité de contenu allant du texte simple à la page web, aux documents multimédias, à une géolocalisation ou encore à des canaux de communication. Ils constituent ainsi un trait d’union entre activité pédagogique et ressources.

Une synthèse de cours évolutive

Utilisés à l’issue d’une séquence, les codes QR permettent de concilier la synthèse et le développement. Le polycopié évolue et il est avantageux de disposer d’un support papier qui est synthétique (pour des raisons économique et pédagogiques). Ils peuvent constituer un outil de révision interactif. L’intérêt étant de jouer sur le décalage temporel entre le moment où l’élève lit la question et le moment où le code scanné ou flashé donne la bonne réponse.

Le code QR est également utilisé dans le cadre du "contrat pédagogique" de la classe à l’aide d’un syllabus interactif. Outil d’explicitation de la démarche pédagogique de l’enseignant à destination des élèves, le syllabus favorise la transparence du contrat, donne du sens aux apprentissages et permet aux élèves d’être acteurs de leur formation. L’utilisation du code QR permet au syllabus d’être interactif (exemple de syllabus interactif).
